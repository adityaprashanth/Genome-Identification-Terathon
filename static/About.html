Genome

ABOUT:

The human genome is the complete set of DNA instructions in a human cell, which is made up of 23 pairs of chromosomes in the nucleus and a small DNA molecule in the mitochondria. The genome contains protein-coding DNA sequences and non-coding DNA, which includes DNA that codes for non-translated RNA, promoters, structural DNA, and transposable elements. The human genome contains 3 billion DNA building blocks, or nucleotides, which is enough to fill about 1,000 telephone books if each nucleotide is represented by one letter.
